Welcome on my Learning CSS website

L'objectif des sessions de travail LEARNING CSS est de permettre aux étudiants d'instaurer une autonomie dans l’acquisition des techniques et des nouveautés en CSS.

Sur les trois dernières années CSS s’est dotée de nouveautés et d'innovations qui sont à la fois excitantes et assoient la CSS comme une discipline incontournable pour tout professionnel du Web.

About CSS

Lorsque vous pensez au HTML et au CSS, vous les imaginez probablement comme un ensemble. Mais pendant des années après que Tim Berners-Lee a créé le World Wide Web en 1989, les CSS n'existaient pas. Le plan initial pour le web n'offrait aucun moyen de styliser un site web.
