Learning CSS

L'objectif des sessions de travail LEARNING CSS est de permettre aux étudiants d'instaurer une autonomie dans l’acquisition des techniques et des nouveautés en CSS. Sur les trois dernières années CSS s’est dotée de nouveautés et d'innovations qui sont à la fois excitantes et assoient la CSS comme une discipline incontournable pour tout professionnel du Web.
